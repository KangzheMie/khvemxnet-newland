「这里是新大陆发现部的档案馆, 用于陈放我们在星海中观察和发掘到的知识. 这些知识作为笔记留存在这里, 是我们继续在宇宙中探索的力量.」

url：https://newland.khvemx.work/

## 开发现状

新大陆发现部 2.x 开发项目，已经上线发布并部署在了服务器上。

2.x 版本开发的基础原则：将后端、前端、内容管理分为三个不同的工程各自独立开发。

- 后端：使用 strapi 搭建后端内容管理系统，提供只读 api 接口供前端使用
- 前端：只负责对 api 获取到的文本内容渲染加工 
- 内容管理：出于安全性，新大陆的所有博客文本全部保存在本地计算机，通过内容管理软件实现内容更新
